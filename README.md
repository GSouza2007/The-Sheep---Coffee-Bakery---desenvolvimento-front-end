# The Sheep – Coffee & Bakery

Site estático de uma cafeteria fictícia chamada **The Sheep**, especializada em cafés e folhados artesanais.

## Tecnologias

- HTML5
- CSS3
- **Sem JavaScript**

## Estrutura de Páginas

- `index.html` – Página inicial, apresentação da cafeteria.
- `about.html` – História, proposta e ambiente da casa.
- `menu.html` – Cardápio em formato de menu, com categorias e preços.
- `pastries.html` – Destaque para folhados e produtos de vitrine.
- `contact.html` – Contatos, links para iFood e Instagram, endereço com mapa do Google Maps.

## Layout e Requisitos

- Menu de navegação superior fixo com até 5 links.
- Logo da cafeteria no canto superior esquerdo (link para a home).
- Margens laterais de 15% para o conteúdo principal.
- Design responsivo simples, focado em legibilidade.

## Como executar

Basta abrir o arquivo `index.html` em qualquer navegador moderno  
(dando dois cliques ou via “Abrir com → Navegador”).
